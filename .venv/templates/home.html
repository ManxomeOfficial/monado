Changelog! [Release: v1.0.0]

What's Included?

As the first official release, features included are the following:
- Global Chatroom that all users are able to access
- The ability to change your avatar and biography in your profile page
- The ability to send, receive, accept, and deny friend requests
- The ability to create new direct message chatrooms with any number of friends
- Live chat featuring websockets, new messages make a ping sfx!
- Image attachments, able to be sent with or independant of text messages
- All messages come equipped with date and time stamps, visible by right-clicking on a message

If you have any features you want to see added, please submit them in the Suggestion Box
